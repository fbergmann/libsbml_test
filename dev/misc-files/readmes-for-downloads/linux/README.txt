                         l i b S B M L   4.3
		Installers for Linux operating systems

                        Released 4 March 2011

IMPORTANT: Please uninstall any existing copy of libSBML located in
the same destination before you install libSBML using this installer,
or you may end up with mixed versions of different libSBML components
on your system.

This directory contains installers for precompiled versions of libSBML
for Linux systems.  All copies were built using the libxml2 XML parser
library.  The installer provides only the core libSBML library (C/C++
interface); it does not include the other language bindings.

Please navigate to either the 32-bit or 64-bit folders, to match the
version of the operating system you are using.

For additional information about this version of libSBML, please see
the files in the "docs" subdirectory of this installer or visit the
libSBML website at 

  http://sbml.org/Software/libSBML

Please report problems with libSBML using the tracker at

  http://sbml.org/Software/libSBML/issue-tracker

Thank you for your interest in libSBML!

The LibSBML Team.
